- **项目介绍:通过传入歌曲/FM音频的播放地址返回其对应的歌曲/音频下载地址**
- **爬虫文件:Spiders目录下的各个python文件**
- **运行方式:运行Spiders目录下main.py文件即可**
## :dolphin:运行环境
**Version: Python3**
